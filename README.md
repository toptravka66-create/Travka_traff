## ⚙️ Customization

To customize the landing page:

1. **Change Colors**: Update hex values in the `<style>` section
   - Green: `#00ff88`
   - Purple: `#9d00ff`

2. **Update Content**: Edit text in the HTML sections
   - Logo, headlines, features, descriptions
   - Telegram link: `https://t.me/soprano67_1`

3. **Modify Animations**: Adjust `@keyframes` in the CSS

## 🚀 Deployment

The site is automatically deployed via GitHub Pages:

1. Push changes to the `main` branch
2. GitHub automatically builds and deploys to the live URL
3. Changes appear within 2-3 minutes

No additional setup required!
